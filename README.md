## Sobre o Projeto

Portfolio pessoal desenvolvido do zero com HTML, CSS e JavaScript puro — sem frameworks, sem dependências de build. Cada detalhe foi pensado para transmitir qualidade técnica e atenção ao design: cursor animado personalizado, transições suaves, sistema de internacionalização PT/EN, e animações de entrada que guiam o olhar do visitante.

## Funcionalidades

- **Cursor animado** com anel de rastreamento e efeito de hover em elementos interativos
- **Barra de progresso de leitura** fixada no topo da página
- **i18n integrado** — alternância instantânea entre Português e Inglês sem recarregar a página
- **Animação de entrada** no hero com slide-up nas letras do nome
- **Reveal on scroll** com `IntersectionObserver` em todas as seções
- **Barras de proficiência** animadas ao entrar na viewport
- **Acordeão de experiências** clicável com transição suave de `max-height`
- **Marquee infinito** com tecnologias rolando horizontalmente
- **Nav com blur** que aparece ao rolar a página
- **Design responsivo** com breakpoints para mobile
- **Orbs flutuantes** no hero com animação CSS `keyframes`
